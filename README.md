# Frontend Setup Scipts

Bash scripts to automatically install dependencies I commonly use such as tailwindcss and shadcn ui
(Assuming the frontend is a vite with react + typescript enviornment)

## Usage Using Git Bash
Tailwind installation: Installs Tailwind dependencies and updates necessary files 
```bash
./addtailwind.sh ./vite-project-name
```

Shadcn installation: Installs Tailwind and Shadcn dependencies and updates necessary files 
```bash
./addshadcn.sh ./vite-project-name
```

ReactRouter installation: ReactRouter dependencies and updates necessary files 
```bash
./addreactrouter.sh ./vite-project-name
```
